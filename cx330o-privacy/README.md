# cx330o Privacy — GDPR/CCPA 合规套件

Klaro.js + GDPR Analyzer 集成，全面的数据隐私合规方案。

## 功能

- Cookie 同意管理（Klaro.js，自动拦截未授权脚本）
- 多法规支持（GDPR / CCPA / LGPD / POPIA）
- 数据主体权利请求（DSAR）处理
- 隐私政策自动生成
- 合规审计日志
- 自动合规扫描

## 集成方式

已内置到 Dashboard，Klaro.js 配置在 `cx330o-dashboard/public/index.html` 中。
GDPR Analyzer 可独立运行合规扫描。
